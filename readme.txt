Plugin dodający przycisk autokorekty w polach tekstowych.

== Description ==

Plugin bazujący na narzędziu do autokorekty tekstu w języku polskim [iKorektor](https://ikorektor.pl).
Dodaje do każdego pola tekstowego na stronie (również w panelu administracyjnym) przycisk autokorekty.
Szczegółowo sposób działania wtyczki został opisany na stronie [https://ikorektor.pl/pluginy](https://ikorektor.pl/pluginy).

Po instalacji wtyczki wymagana jest aktywacja po stronie aplikacji. W tym celu należy zgłosić nazwę strony (domeny) w [formularzu kontaktowym](https://ikorektor.pl/kontakt).
